How Javascipt works:

Javascipt is an interpreted and single-threaded language. Everything executed in Javascript inside execution context. We can assume execution contexr as two parts: 1)memory 2) code . Memory is also known as variable environment, where everything stored as keys and values, and code is known as thread, where each execution occured at a time. It has a single call stack. it stores data on top when we define a function. when we Call the function we get it from that stack.

Event loop in Javascipt:

In JS event-loop is a thing by which we can explain the process of executing code. It has three important parts 1)Heap 2)Queue 3)Stack .Heap is a large memry region where objects are allocated. Queue is a list of messages yet to be processed. Like when we call setTimeout and fetch related thing it will be placed on queue. For the function execution we use stack where a frame is created on function definition and this frame is called on function call. stack is placed on top of each other.
